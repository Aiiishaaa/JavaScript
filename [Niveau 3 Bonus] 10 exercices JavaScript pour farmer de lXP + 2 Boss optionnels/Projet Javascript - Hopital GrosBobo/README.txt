# Hopital GrosBobo
Animer une page en javascript

# Contexte :
L'hôpital "Grosbobo" de Springfield souhaite développer un outil d'apprentissage pour ses infirmiers et internes. 
En effet, leurs erreurs aussi grossières que régulières nuisent fortement à la réputation de l'établissement. 
Ce site aura pour vocation de former le personnel pour ne plus qu'il se trompe lors d’interventions pour soigner les patients.

# Projet :
Vous devez réaliser une page web, montrant les différentes parties du corps comme dans l'exemple donné et la description de celles-ci.
Vous pouvez travailler en binôme.

# Contraintes :
Vous devez suivre la maquette fournie dans le fichier resultat.png.
L'image du corps est directement mise à disposition par le service des archives de l'Hôpital GrosBobo et est à récupérer ici : corps.png. Lorsque vous cliquerez sur l'un des disques (à disposer sur le corps, il s'agit de div avec du CSS), le texte descriptif (que vous irez cherchez sur wikipedia par exemple) s'affichera dans la colonne de droite. 

# Consignes :
● Utilisation des attributs HTML "class" et "id" dans le code HTML

● Utilisation de la police Arvo (voir sur GoogleFont)

● Utilisation des balises et attributs HTML sémantiques

● Séparation du HTML, du CSS et du JavaScript pour un code propre et lisible

● Utilisation des media-queries pour obtenir un résultat responsive

● Possibilité d'utiliser jQuery (aucune obligation)

● Vous pouvez bien évidemment choisir la couleur de fond ainsi que les styles pour améliorer la mise en page.
