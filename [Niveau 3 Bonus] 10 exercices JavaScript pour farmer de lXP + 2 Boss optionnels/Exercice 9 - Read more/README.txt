# Objectif
_Afficher un bouton "Lire plus" permettant d'afficher un contenu d'abord masqué._

# Enoncé
1) Créer un document HTML contenant une première div qui contiendra la version courte du texte, dans notre cas simplement "Lorem ipsum ..." (chercher sur google un générateur de "lorem ipsum")
2) Ajouter ensuite un bouton "Lire plus"
3) Ajouter une seconde div contenant l'intégralité du texte "Lorem ipsum" (Si vous utiulisez Sublime Text essayez "lorem"+tab). Cette div sera masquée à l'ouverture de la page ! (propriété css display:none)
4) Lorsque l'utilisateur clique sur "Lire plus", la première div (texte court) disparait, de même que le bouton "Lire plus" et c'est la seconde div, le texte long qui s'affiche. Vous devez donc gérer les modifier de propriété css de ces éléments.
